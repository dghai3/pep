# DTF print-prep pipeline - pinned dependencies
# Tested with Python 3.10 / 3.11.  (basicsr/realesrgan can be fiddly on 3.12.)
#
# IMPORTANT - PyTorch install depends on your hardware. Do ONE of the following
# BEFORE `pip install -r requirements.txt`, because the right torch wheel must
# come from the right index:
#
#   Windows / Linux + NVIDIA GPU (e.g. RTX 3070), CUDA 12.1:
#     pip install torch==2.3.1 torchvision==0.18.1 --index-url https://download.pytorch.org/whl/cu121
#
#   Apple Silicon Mac (M1/M2/M3/M4) - MPS acceleration, default wheels:
#     pip install torch==2.3.1 torchvision==0.18.1
#
#   CPU-only (any OS):
#     pip install torch==2.3.1 torchvision==0.18.1 --index-url https://download.pytorch.org/whl/cpu
#
# Then: pip install -r requirements.txt
# (torch/torchvision are listed below too, so a plain `pip install -r` also works
#  and will pull the default CPU/MPS wheels if you skipped the step above.)
